# 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on a dataset to understand data patterns, correlations, distributions, outliers, and data quality issues.

The analysis helps in:
- 📊 Understanding dataset structure
- 📈 Finding trends & correlations
- ⚠️ Detecting outliers
- 🧹 Checking missing values & datatype issues
- 📉 Visualizing numerical & categorical features
